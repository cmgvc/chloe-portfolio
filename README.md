# Portfolio 2026 ⭐

Welcome to the digital sanctuary of **Chloe Gavrilovic**. This isn’t just a portfolio; it’s an immersive, lo-fi aesthetic experience built at the intersection of web engineering and 3D artistry.

---

## The Tech Stack
Crafted with precision, designed for performance:

* **Core:** Built on React and styled with Tailwind CSS for a seamless, modern flow.
* **3D Engine:** Powered by `react-three-fiber` and `Three.js`, turning the browser into a canvas (TDB).
* **The Artifacts:** 3D models sculpted in Blender and exported as GLTF (TBD).
* **Visual Fidelity:** Textures baked in Cycles (JPG exports). By baking lighting and shadows directly into the textures, we achieve cinematic realism without the overhead of real-time calculations (TBD).
